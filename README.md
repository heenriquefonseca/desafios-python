---

## Como executar

Clone o repositório:

```bash
git clone https://github.com/heenriquefonseca/desafios-python.git
```

Entre na pasta:

```bash
cd desafios-python
```

Execute qualquer arquivo com:

```bash
python nome_do_arquivo.py
```
